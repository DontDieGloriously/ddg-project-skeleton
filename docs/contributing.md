# Contributing

Thank you for your interest in contributing!  This project is open to community contributions.  To get started:

1. Review the [Code of Conduct](../CODE_OF_CONDUCT.md) and [Contributing guidelines](../CONTRIBUTING.md) to understand our expectations.
2. Check existing issues and pull requests to see if your idea or bug report has already been filed.
3. Use the issue templates to report bugs or request features and .
4. Fork the repository, create a feature branch and open a pull request when your change is ready.  Our pull request template will guide you on what information to include.

If you have questions or need help, start a discussion in the Don't Die Gloriously **Discord**: https://discord.dontdiegloriously.com
