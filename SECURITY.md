# Security Policy

Thank you for your interest in the security of this project.  We take security seriously and appreciate your efforts to responsibly disclose vulnerabilities.

## Supported Versions

We maintain the latest stable release of each project.  When a new major or minor version is released, we will provide security fixes only for the most recent release.

## Reporting a Vulnerability

To report a security vulnerability, please **do not file a public issue**.  Instead, email us at <security@dontdiegloriously.com> with a description of the issue and steps to reproduce it.  We will respond as quickly as possible and work with you to verify the vulnerability and determine a fix【173615496502311†L468-L476】.
