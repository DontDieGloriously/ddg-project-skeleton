# Getting help

Thank you for using our software.  If you need assistance, please consider the following options:

* **Search existing issues** – someone may have already reported your problem.  Browse and comment on existing issues where appropriate.
* **Start a discussion** – for general questions/discussion, please use our **Discord**: https://discord.dontdiegloriously.com.
* **Open an issue** – if you have found a bug or have a feature request, please use the appropriate issue template.  Provide as much detail as possible to help us understand your request.

For security issues, please **do not** file a public issue.  Instead, follow the instructions in [SECURITY.md](SECURITY.md).
